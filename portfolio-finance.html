Project Overview

In this project, I utilized statistical learning methods to build a valuation model for companies.

Methodology: Using R, I applied Lasso Regression to handle multicollinearity and perform feature selection among hundreds of financial indicators. The model successfully identified the key drivers of company value and achieved a high prediction accuracy on the test set.

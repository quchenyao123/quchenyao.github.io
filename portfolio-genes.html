Cancer Classification with High-Dimensional Data

Back to Projects

Project Overview

This project focuses on analyzing high-dimensional gene expression data to classify cancer types.

The Challenge: Gene data often contains thousands of variables (genes) but very few samples (patients), making traditional statistical methods difficult to apply.

Solution: I implemented dimensionality reduction techniques and classification algorithms to accurately identify cancer subtypes, helping to pave the way for more personalized medical treatments.

Project Info
